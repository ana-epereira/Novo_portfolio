# Portfólio de Análise de Dados

Este é um website de portfólio estático preparado para GitHub Pages. Ele apresenta sua experiência em análise de dados, habilidades, projetos e informações de contato.

## Estrutura

- `index.html` - página principal do portfólio
- `styles.css` - estilos visuais do site
- `script.js` - navegação responsiva do menu

## Publicação no GitHub

1. Crie um repositório no GitHub.
2. Envie estes arquivos para o repositório.
3. No GitHub, vá em **Settings > Pages**.
4. Selecione a branch `main` (ou `master`) e a pasta `/root`.
5. Salve e aguarde alguns minutos até o site ficar disponível.

## Personalização

- Altere o texto de contato em `index.html`.
- Adicione seus projetos reais e links para o GitHub.
- Substitua `seuemail@exemplo.com`, `linkedin.com/in/seu-nome` e `github.com/seu-usuario`.
